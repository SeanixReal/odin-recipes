# Odin Recipes

A collection of delicious recipes with a clean, modern web interface. This project showcases HTML and CSS skills while providing easy access to various recipes.

## Features

- Clean and responsive recipe pages
- Easy navigation between recipes
- Beautiful typography using Google Fonts (Montserrat)
- Professional styling with custom CSS

## Recipes Included

1. **Creamy Pepperoni Chicken** - A quick and easy dish with a rich, tangy sauce
2. **Potato Latkes** - Classic crispy potato pancakes perfect for any occasion
3. **Yorkshire Pudding** - Light, fluffy, and golden-brown puddings
